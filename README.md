# USER PLUGINS

## Custom Profile
duh. change username, pfp, banner, badges, bio, pronouns, etc.

## FakeDM
adds a button in dms that lets you pick between who sends the message, and if it's a message or call.

## FakeEdit
right click a message, then click "fake edit". it'll make the message say edited when it really isnt - appears the same, but has potential to mess with people.
