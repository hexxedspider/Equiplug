# USER PLUGINS

## AutoReact
set up userId:emoji pairs in settings, then ure you're good. configurable delay asw.

## AutoTranslateNightcord
plugin for nightcord, which got outted for being both a key logger, and token logger. i dont remember if it was both, or just one or the other, but either way, both are bad lol. the plugin is safe, since it was the app that was infected. this just provides translations for diff languages.

## Custom Profile
duh. change username, pfp, banner, badges, bio, pronouns, etc.

## FakeDM
adds a button in dms that lets you pick between who sends the message, and if it's a message or call.

## FakeEdit
right click a message, then click "fake edit". it'll make the message say edited when it really isnt - appears the same, but has potential to mess with people.

## FakeFriends
fake having someone added, or fake a friend request. right click on their profile to do this.

## FakeSwitcher
right click on a user, then scroll until you see it. then it'll show under "switch accounts" in the profile panel.

## FakeSystemMessage
inject fake system messages into channels. doesn't work perfectly, but like 97% plus it's cool.

## FakeTyping
adds a "/infinitype" command that toggles between on/off. makes it look like you're typing forever.

## FakeVoice
appears muted, deafened, or both, while being neither... meaning you can appear to not be able to hear, but can.

## Impersonate
similar to fakeDM, but better. adds a "/impersonate" command that can be modified to have any user, attachment, how long until it sends, etc., plus it works in servers - fakeDM does not work in servers.

## Mimic
set a target user ID. whenever they send a message, you automatically send the same message in whatever channel you're currently looking at. strips pings by default so you don't double ping.

## StickVoiceChannel
right click on a voice channel to see the option to force yourself back into a channel by rejoining. have to toggle it off in order to be able to leave though.
